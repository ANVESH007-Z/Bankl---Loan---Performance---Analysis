# Loan KPI Queries

## Purpose

This SQL script calculates key performance indicators (KPIs) for the Bank Loan Analysis project.

## KPIs Covered

### Loan Applications

* Total Loan Applications
* Month-to-Date (MTD) Loan Applications
* Previous Month-to-Date (PMTD) Loan Applications

### Funded Amount

* Total Funded Amount
* MTD Funded Amount
* PMTD Funded Amount

### Amount Received

* Total Amount Received
* MTD Amount Received
* PMTD Amount Received

### Interest Rate Analysis

* Average Interest Rate
* MTD Average Interest Rate
* PMTD Average Interest Rate

### Debt-to-Income (DTI) Analysis

* Average DTI
* MTD Average DTI
* PMTD Average DTI

## Database

* PostgreSQL

## Table Used

* financial_loan

## Objective

To measure lending performance, monitor loan portfolio health, and provide insights for Power BI dashboard reporting.
