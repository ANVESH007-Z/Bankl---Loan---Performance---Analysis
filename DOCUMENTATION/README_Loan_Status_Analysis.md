# Loan Status Analysis

## Purpose

This SQL script analyzes loan performance based on loan status categories. 
It provides a summary of applications, funded amounts, repayments, interest rates
and debt-to-income ratios for each loan status.

## Metrics Covered

### Loan Status Summary

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average Debt-to-Income Ratio (DTI)

### Month-to-Date (MTD) Analysis

* MTD Funded Amount by Loan Status
* MTD Amount Received by Loan Status

## Database

* PostgreSQL

## Table Used

* financial_loan

## Loan Status Categories

Common loan statuses include:

* Fully Paid
* Current
* Charged Off

## Business Objective

This analysis helps monitor loan portfolio performance across different loan statuses. It enables stakeholders to assess repayment trends, funding allocation, loan quality, and overall portfolio health.
